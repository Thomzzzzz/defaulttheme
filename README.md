# Default Theme

Deze repository wordt gebruikt om een up to date thema voor WordPress te hebben.

### Standaard documenten
Elke thema bevat een aantal standaard documenten, namelijk:
*   index
*   header
*   footer
*   404
*   functions
*   archive
*   page
*   single
*   search
